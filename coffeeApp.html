Koffee

An app prototype which would allow users to order and collect coffee from their phones. This would potentially speed the process of ordering items from a coffee shop, and would avoid long excessive queues at the till.

Designed using Sketch App and built using Ionic Framework for Cordova. The skills used include: HTML, CSS, JavaScript (Angular)
