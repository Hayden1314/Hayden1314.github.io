Fish And Chips App

A proposal app idea for a local fish and chip shop. Similar to Just Eat or Deliveroo, the aim of this application is to allow users to order their food on their mobile devices. Compared to existing delivery apps, this would only be specifically for this shop.

Designed using Sketch App and built using Ionic Framework for Cordova. The skills used include: HTML, CSS, JavaScript (Angular)
